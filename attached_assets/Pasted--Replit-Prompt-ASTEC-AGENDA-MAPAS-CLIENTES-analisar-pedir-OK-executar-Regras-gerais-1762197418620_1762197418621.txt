1.2 Endpoints (Express + TS)

Activities (Agenda):

GET /api/activities?userId=&dateFrom=&dateTo= → RBAC: admin vê todos; assistente só os seus.

POST /api/activities → cria (para F-001/F-002).

PUT /api/activities/:id → atualiza horários (drag/resize).

DELETE /api/activities/:id

Activity Types:

GET/POST/PUT/DELETE /api/activity-types

Technician status/telemetria (F-003):

GET /api/technicians/status → { technicianId, status, gpsStatus, lastLocation{lat,lon,updatedAt}, battery, accuracy, device, androidVersion, appVersion }

GET /api/technicians/:id/last-location

WebSocket /ws/locations → push { technicianId, lat, lon, accuracy, battery, gpsStatus, connectionStatus, updatedAt }

Clients & sites (F-004/F-005):

GET /api/map/clients?group=&segment=&search= → pontos para mapa/cluster

GET /api/clients (paginado + filtros)

POST /api/clients

PUT /api/clients/:id

POST /api/clients/import (upload .xlsx/.csv)

POST /api/geo/parse-maps-url → recebe URL Google Maps e retorna {address parts, lat, lon}
